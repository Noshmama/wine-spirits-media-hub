How I Built This

Wine & Spirits Media Hub is a fully automated content platform built without frameworks — just HTML, CSS, and JavaScript.

Data Pipeline: Node.js scripts query the YouTube Data API and iTunes Search API weekly via GitHub Actions, pulling fresh videos and podcasts across 24 categories. Content is deduplicated, tagged by beverage type and region, and filtered for language and relevance — all without manual intervention.

Performance: The app renders 1,000+ cards once on page load, then filters instantly via CSS display toggling — no DOM rebuilding. Images lazy-load via IntersectionObserver, and pagination keeps initial render fast. Data files are trimmed to exclude unused fields, cutting download size by 75%.

Stack: HTML/CSS/JS, Node.js, YouTube Data API v3, iTunes Search API, RSS parsing, GitHub Actions, GitHub Pages.
